# 📝 Todo App (Local Storage)

A simple and responsive Todo App built using HTML, CSS, and JavaScript. This project allows users to manage daily tasks with data stored in the browser using Local Storage, so tasks remain available even after refreshing the page.

## 🚀 Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ✏️ Edit existing tasks
- 🗑️ Delete individual tasks
- 🧹 Clear all tasks
- 💾 Data saved using Local Storage
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 💡 How It Works

1. Enter a task in the input field.
2. Click the **Add** button.
3. Your task is instantly saved in Local Storage.
4. complete, or delete tasks anytime.
5. Even after refreshing the page, your tasks remain saved.
